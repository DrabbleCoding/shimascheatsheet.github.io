Tables

Tag Name Description Example
<table> Table wrapper element that encompases all HTML tables 
                            <table>
                              <tr>
                                ...
                              </tr>
                            </table>
<tr> Table Row adds rows to the table, without any data 
                        <table>
                          <tr>
                            <td>cell 1 data</td>
                            <td>cell 2 data</td>
                          </tr>
                        </table>
<td> Table Data adds cell of data to the table, nested inside a <tr> element see example above
<thead> Table Head defines head of table columns within a table row (settings could be applied using css ruling with this tag, separate from tr) 
                        <table>
                          <thead>
                            <tr>
                              <th>heading 1</th>
                              <th>heading 2</th>
                            </tr>
                          <thead>
                          <tbody>
                            <tr>
                              <td>col 1</td>
                              <td>col 2</td>
                            </tr>
                          </body>
                        </table>
<th> Table Heading adds titles to rows and columbs of a table, must be enclosed within a <tr> element 
                        <table>
                          <tr>
                            <th>column 1</th>
                            <th>column 2</th>
                          </tr>
                          <tr>
                            <td>data 1</td>
                            <td>data 2</td>
                          </tr>
                        </table>
<tbody> Table Body semantic element that contains all table data other than the table heading and the footer (<table> cannot have both <tbody> and <tr> as dirent children 
                        <table>
                          <tr>
                            <th>column 1</th>
                            <th>column 2</th>
                          </tr>
                          <tr>
                            <td>data 1</td>
                            <td>data 2</td>
                          </tr>
                        </table>
<tfoot> Table Footer uses table rows to give footer content or to summarize content at the end of a table.
colspan column spreader indicated how many columns that particular cell should span within the table. This attribute could be applied to both a table header <th> and a table data <td>. The value is set to 1 by default, and will take any positive integer between 1 and 1000
rowspan row speader similar to colspan, it indicates how many rows a cell should span within the table. It could be applied to both a table header or a table element.
